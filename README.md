# 概要  
LinkinPark Platformなどのように床や壁でUIが隠れてしまう場合に透過して表示させるMODです  
今のところalpha版で、Bloomを付けている場合影響を受けて眩しくなってしまう点、本来のUIより若干解像度が低くなってます。
